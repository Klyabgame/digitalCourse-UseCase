# DigitalCourse-UseCase

**DigitalCourse** es una aplicación de ventas online que se especializa en la venta de cursos de programación. Este proyecto tiene como objetivo **generar los casos de uso detalladamente** para ayudar en la planificación y desarrollo de la plataforma. Los casos de uso proporcionarán una descripción clara de cómo los usuarios interactúan con la plataforma y las funcionalidades que ofrece.

## Descripción

La aplicación **DigitalCourse** permite a los usuarios **comprar y vender cursos de programación** en línea. A través de la plataforma, los estudiantes pueden navegar por diversos cursos de tecnología, programación y desarrollo, mientras que los instructores pueden ofrecer sus propios cursos.
